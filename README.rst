------------------
Build Instructions
------------------

Pip install (into a virtual environment)::

    pip install dmpr

or::
    
    pip install git+https://github.com/ScottWales/dmpr

or (for development)::

    git checkout git+https://github.com/ScottWales/dmpr
    cd dmpr
    pip install -e '.[dev]'
